Open Web Application Security Project (OWASP)

spring security:
--------------------------
module in a spring ecosystem which takes care of the security aspect of your application

security:
-------------
authentication and authorization features

1. provides secure authentication and authorization for users
2. Supports form login, JWT, OAuth2, LDAP and OAuth2 with Keycloak
3. protects your app against CSRF, CORS, and session attacks
